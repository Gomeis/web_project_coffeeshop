# Triple Peaks Coffee Shop

## Descrição do projeto

Este projeto consiste em uma landing page para a cafeteria fictícia Triple Peaks, desenvolvida como parte de um projeto de HTML e CSS avançado.

A página apresenta:

- um cabeçalho com navegação, texto introdutório e imagem principal;
- uma seção de receitas com vídeos incorporados;
- uma seção de reserva com formulário;
- um rodapé com links para redes sociais.

## Tecnologias utilizadas

- HTML5
- CSS3
- Metodologia BEM
- Flexbox
- Grid Layout
- Normalize.css
- Estrutura de arquivos organizada por blocos CSS

## Planos de melhoria do projeto

No futuro, gostaria de melhorar este projeto com:

- responsividade para diferentes tamanhos de tela;
- animações suaves em botões e links;
- validação mais avançada no formulário de reserva;
- integração com backend para envio real das reservas.
